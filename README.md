# ToDoList
That's a simple web app to create and delete simple toDo tasks;
The project is not complete and have some bugs (especially in front-end);
This project is divided into two repositorys, this one for the front-end end one for the back-end;
The repository for the back-end can be found in this [link](https://github.com/DevMoitinho/toDoListBack).

## Languages and frameworks
- Front-end:
    - React js
- Back-end:
    - Java
    - SpringBoot

## Configuração

Crie um arquivo `.env` na raiz do projeto:
VITE_API_URL=http://localhost:8080
